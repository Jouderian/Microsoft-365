# analiseConfiguracoesDeSeguranca.ps1

> **Sinopse**: Executa o script Orca (https://github.com/cammurray/orca) que analisa as configuração recomendada do Microsoft Defender para M365

## Descrição
O script se conecta ao ambiente do Microsoft 365, executa o script Orca e gera um relatório em HTML com as configurações recomendadas.
